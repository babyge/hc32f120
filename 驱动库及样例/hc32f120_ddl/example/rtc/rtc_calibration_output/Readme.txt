功能描述
================================================================================
本样例展示Rtc模块时钟误差补偿及1Hz输出功能。

说明：
本样例设置Rtc为24小时时间格式，通过按键SW1和SW2增减补偿值，并设置端口P30为1Hz输
出端口，配置1秒周期中断，LED_R每秒闪烁。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32F120_LQFP44_080_V10

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）将示波器连接到P30端口，测量频率为1Hz；
4）连续点击按键SW1或SW2，观察测量的频率的变化；
5）同时观察LED_R持续闪烁。
