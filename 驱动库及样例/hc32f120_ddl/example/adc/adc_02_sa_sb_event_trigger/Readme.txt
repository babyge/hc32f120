功能描述
================================================================================
本例程主要实现了用外部触发源触发ADC的序列A和序列B转换，其中，序列A的触发条件为
引脚ADTRG（本例为P10）下降沿（由高到低，并在低保持1.5*4个HCLK周期），序列B的触
发条件为序列A采样转换结束后所产生的事件EVT_ADC_EOCA。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的调试功能，设置断点，全速运行；
3）拉低P10，触发序列A动作，继而由序列A结束所产生的事件EVT_ADC_EOCA，触发序列B动作；
4）断点查看程序执行情况。


================================================================================
注意
================================================================================
ADC有三个外部触发引脚（P10，P50和P60），它们的功能是一样的，三个引脚与序列A和序列B无
对应关系，其中一个序列选择了外部引脚作为触发源，另外一个序列就不可以再选择外部引脚作为
触发源。如果选择内部事件作为触发源，两个序列不可设置相同的事件作为触发源。
