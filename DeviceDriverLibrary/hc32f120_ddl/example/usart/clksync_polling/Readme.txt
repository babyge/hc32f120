功能描述
================================================================================
本样例主要展示USART外设配置为时钟同步模式时通过轮询方式收发数据。
说明：
无

测试环境
================================================================================

---------------------
STK_HC32F120_LQFP44_080_V11

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）需两个测试板，按照如下方式连接
   主机:UART2      从机:UART2
      TXD-------------RXD
      RXD-------------TXD
       CK-------------CK
      GND-------------GND
2）打开工程clksync_polling, 配置宏定义如下，重新编译，下载代码主机：
   #define CLKSYNC_DEVICE_MODE             CLKSYNC_MASTER_MODE
3）修改如下宏定义，并重新编译，下载代码从机。
   #define CLKSYNC_DEVICE_MODE             CLKSYNC_SLAVE_MODE
4）分别重新上电
5）按从机 SW2按键
6）按主机 SW2按键
7）主机和从机 绿色LED点亮，表示两个板子收发数据一致，符合预期
