功能描述
================================================================================
本样例主要展示CTC通过CTCREF校准内部高速振荡器HRC。

说明：
1、首先设置CTC_CR1 TRMVAL为0x21，使内部高速时钟频率在48±0.24MHz范围外
2、然后输入CTCREF引脚20MHz时钟，通过CTCREF校准内部高速时钟48MHz，校准偏差为0.5%，
   目标频率为48±0.24MHz范围内

================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32F120_LQFP44_080_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
串口助手

================================================================================
使用步骤
================================================================================
1）准备两个测试板
   测试板A：用于展示校准内部高速时钟
   测试板B：输出XTAL时钟，作为测试板A CTCREF时钟源
2）按照如下方式，连接两个测试板引脚(J11_P15: J11表示接插件丝印，P15表示信号丝印;其
   他表示与此相同)
   测试板A          测试板B
   J11_P11 <----->  J11_P15
   ISP_GND <----->  ISP_GND
3）测试板A J11_P15与示波器相连
4）按照如下，修改main.c文件宏定义
   #define PROJCET_FUNCTION                FUNCTION_GENERATE_CTCREF_CLK
5）重新编译，下载代码至测试板B
6）按照如下，修改main.c文件
   #define PROJCET_FUNCTION                FUNCTION_TRIMMING_HRC_CLK
7）按照如下，修改文件hc32f120_icg.h
   #if ICG0_HRC_HARDWARE_START == ICG_FUNCTION_ON
       #define ICG0_HRC_REG_CONFIG                 ((uint32_t)ICG0_HRC_REQS)
   #else
       #define ICG0_HRC_REG_CONFIG                 ((uint32_t)ICG_HRC_FREQ_48MHZ)
   #endif
8）重新编译，下载代码至测试板A
9）打开串口助手软件，配置端口如下参数
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
10）观察示波器，查看波形频率为48±0.24MHz范围外
11）按测试板SW2按键
12）查看RGB LED灯状态
    若RGB绿色LED灯点亮，表示Trimming成功，查看串口助手打印信息
    若RGB红色LED灯点亮，表示Trimming失败
13）观察示波器，查看波形频率为48±0.24MHz范围内，符合预期
14）按测试板NRST按键，复位开发板
15）重复步骤10）~14），观察LED灯状态和串口助手打印信息
