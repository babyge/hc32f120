功能描述
================================================================================
本例程实现了CRC16和CRC32的计算和校验。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_HC32F120_LQFP44_080_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的调试功能，断点查看程序执行情况；


================================================================================
注意
================================================================================
CRC_Calculate 和 CRC_Check 函数的最后一个参数表示输入数据的位宽，请严格按照输入数据的
位宽设置该参数。
