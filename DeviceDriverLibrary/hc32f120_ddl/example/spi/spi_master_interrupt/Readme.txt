功能描述
================================================================================
本例程实现了SPI错误中断的配置（包括独立中断配置和共享中断配置）和基本用法。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32F120_LQFP44_080_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的调试功能，断点查看程序执行情况；


================================================================================
注意
================================================================================
1、SPI通信过程中发生如下错误均可触发错误中断：
   （1）过载错误
   （2）奇偶校验错误
   （3）欠载错误
   （4）模式错误
